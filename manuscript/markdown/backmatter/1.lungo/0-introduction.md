## Revisiting JavaScript Allongé

JavaScript Spessore is written for the reader who has read [JavaScript Allongé](https://leanpub.com/javascript-allonge) or has equivalent familiarity with functions and basic use of JavaScript's objects, functions, closures, and the prototype chain.

For those who haven't read JavaScript Allongé or would like a refresher, some of the material from JavaScript Allongé is abbreviated and reproduced in this appendix. We'll discuss how JavaScript is typically used to create objects with methods using functions as constructors.
